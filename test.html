30 ejemplos de sitios web usando el Bootstrap Framework de Twitter

Por Marcos Rivas lunes, marzo 11, 2013
COMPARTE EL POST
Tweet
Post Anterior (J)
Siguiente Post (K)
▲

Twitter además de ser una red social tiene un poderoso framework llamado Boostrap, el cuál ayuda mucho a simplificar el proceso de creación de sitios web. Lo que hace boostrap es definir una serie de reglas CSS para que tu sólo te preocupes por colocar el contenido en tu sitio web y el framework se encargue de irlo diseñando. El framework es más que nada CSS pero la ventaja que tiene es que los nombres de clases son sintácticas, es decir, el nombre de la clase define lo que va a hacer.

El framework es gratuito y se puede usar para fines comerciales, por eso les presento 30 ejemplos de sitios web usando Bootstrap de Twitter, para que vean lo que algunas empresas han logrado de una forma muy simplificada. Un tip, si se fijan, muchas comparten un layout muy parecido, y es que Boostrap nos da una plantilla base para comenzar, y ya después depende de nosotros darle nuestro toque personal.
